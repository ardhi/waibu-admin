Admin for [Waibu MPA](https://github.com/ardhi/waibu-mpa)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install waibu-admin
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```waibu-admin``` in it
